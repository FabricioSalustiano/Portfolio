Sobre Mim

Sou Fabricio Salustiano, desenvolvedor full stack apaixonado por unir tecnologia, arte e propósito. Acredito que cada linha de código pode ir além de resolver problemas — pode criar experiências significativas, gerar impacto positivo e melhorar a vida das pessoas.

Minha trajetória combina sólida experiência técnica, atuando com Angular 18+, PHP Laravel, Java Spring, MySQL, Oracle SQL, WebSockets, microserviços, Kafka, Docker e práticas DevOps, com um olhar cuidadoso para a usabilidade, performance e experiência do usuário. Tenho experiência em arquiteturas limpas (Clean Architecture), código limpo (Clean Code) e automação de processos com pipelines CI/CD.

Mais do que entregar sistemas robustos, busco criar soluções que contem histórias, despertem conexão e sejam capazes de transformar realidades. Para mim, programação é uma ponte entre o humano e o digital — e é nessa interseção que encontro meu propósito. Ao longo da minha carreira, já atuei desde o desenvolvimento de funcionalidades para sistemas legados até a construção de plataformas modernas escaláveis e seguras.

Meu compromisso é entender a fundo cada projeto, planejar com estratégia e executar com excelência, sempre visando entregar valor real para clientes e usuários. Se você busca alguém que una criatividade, técnica e foco em resultados, está no lugar certo.

Processo de Trabalho

Pesquisa e Planejamento

Compreendo as necessidades do cliente e realizo levantamento de requisitos. Analiso as melhores tecnologias e arquiteturas, considerando escalabilidade, segurança e experiência do usuário.

Design e Desenvolvimento

Transformo ideias em soluções digitais modernas, desenvolvendo aplicações web e APIs com Angular, Laravel, Java Spring, Docker e práticas de Clean Code e Clean Architecture.

Entrega e Otimização

Realizo deploy automatizado via pipelines CI/CD, garantindo qualidade e confiabilidade. Após a entrega, acompanho métricas e feedbacks para evoluir continuamente o projeto.
